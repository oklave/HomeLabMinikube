Абстракиция Service позволяет:
1. Стабильно функционировать виртуальному кластеру (ClusterIP)
2. Иметь стабильное DNS-имя


Ключевые компоненты при функционировании kubernetes:

1. Service - абстракция "группа подов + порт" (API-объект в etcd)
2. kube-proxy - Реализует маршрутизацию к бэкендам (на каждой ноде)
3. CoreDNS - Превращает web-svc → ClusterIP (системный под kube-system)
4. Endpoints / EndpointSlice - Список актуальных IP подов за сервисом
